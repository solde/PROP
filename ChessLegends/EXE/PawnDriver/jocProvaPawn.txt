Joc De Prova per: Pawn

Classe a provar : Pawn

Classes addicionals: Board, Piece

Driver:PawnDriver

Funcionalitats:
L'executable presenta un menu d'opcions les quals s'escullen a traves del teclat, prement la tecla numerica que es desitxi. 

Test 1: Get Max: Prova getMax. Comprova el resutlat amb un d'incorrecte
Test 2: Set Max. Prova setMax. Ho fa canviant max per 7 i comprovant si s'ha efectuat el canvi
Test 3: Get Value. Prova getValue. Equipara el resultat amb un'incorrecte
Test 4: Set Value. Prova setValue. Canvia el valor per 7 i comprova si s'ha canviat
Test 5:get_poss_mov. Prova get_poss_mov. Fa tres comprovacions, una amb posició (2,1) i Board buit, un altra amb (5,4) i Board buit, i  finalment (5,4) amb board no buit. Al final de cada test s'imprimeix la llista de moviments.
